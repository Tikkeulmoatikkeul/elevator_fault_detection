# Elevator Fault Detection

### Overview

### Dataset
* source: https://github.com/datasetengineer/ElevatorFaultDetection
* location: high-rise commercial buildings in Tokyo, Japan
* duration: 2020.01 - 2024.12


### Features
1. Core Operation
2. Mechanical
3. Electrical
4. Environmental
5. Usage
6. Maintenance & Historical
7. Advanced Diagnostic
8. Predictive
   - RUL (Remaining Useful Life)
   - Health Indices (Composite Score)
9. Target Labels
   - Fault State (binary)
   - Fault Category (category)
   - Fault Severity (ordinal)
